# AI Software Engineer Agent - Parte 45

# Deployment Agent: Docker, CI/CD, ambienti, release management, monitoring e rollback automatico

## Obiettivo

Definire l'agente responsabile del rilascio del software.

Il Deployment Agent trasforma il codice validato in un prodotto eseguibile.

Gestisce:

* containerizzazione;
* configurazione ambienti;
* pipeline CI/CD;
* deployment;
* monitoraggio;
* rollback.
