# 1. Memory Manager Agent

## Responsabilità

È il punto unico di accesso alla memoria.

Gestisce:

* salvataggio;
* recupero;
* aggiornamento;
* eliminazione;
* ranking.

---

Gli altri agenti non interrogano direttamente il database.

Esempio:

Sbagliato:

```python
vector_db.search()
```

---

Corretto:

```python
memory_manager.retrieve(
"authentication architecture"
)
```

# Principio finale

La memoria non è un archivio.

È il modello mentale del progetto.

Un senior developer non ricorda ogni riga di codice.

Ricorda:

* perché esiste una componente;
* quali sono i vincoli;
* quali errori sono già successi;
* dove intervenire.

Il Global Memory Manager è ciò che permette all'agente di sviluppare un progetto per mesi mantenendo continuità.
