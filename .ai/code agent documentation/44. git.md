# AI Software Engineer Agent - Parte 44

# Git Agent avanzato: branch automatici, commit intelligenti, rollback, merge conflict resolution e code review automatica

## Obiettivo

Definire l'agente responsabile della gestione del ciclo di vita del codice tramite Git.

Il Git Agent non è un semplice wrapper di comandi Git.

Deve comportarsi come un Software Engineer esperto che mantiene:

* ordine del repository;
* tracciabilità delle modifiche;
* qualità dei commit;
* sicurezza del flusso di sviluppo.
