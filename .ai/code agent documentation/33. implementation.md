# Principio finale

La complessità di questo progetto non è generare codice.

Il problema difficile è costruire un sistema che:

* capisca il contesto;
* pianifichi;
* agisca;
* controlli gli errori;
* impari dagli interventi.

La prima versione deve essere un buon "junior developer autonomo".

Le versioni successive lo trasformano in un vero "software engineer".
