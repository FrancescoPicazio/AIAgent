# AI Software Engineer Agent - Parte 52

# Memory Agent avanzato: memoria breve/lunga, episodica, semantica, procedurale e gestione del contesto infinito

## Obiettivo

Definire il sistema di memoria che permette all'AI Software Engineer di mantenere continuità tra sessioni, progetti e decisioni.

La memoria deve permettere all'agente di:

* ricordare conversazioni precedenti;
* comprendere il progetto nel tempo;
* apprendere dalle modifiche;
* evitare errori già commessi;
* mantenere coerenza architetturale.

Il sistema utilizza più livelli.

---

## Short Term Memory

Memoria della sessione corrente.

Contiene:

* messaggi recenti;
* task corrente;
* stato workflow;
* risultati intermedi.

---

Esempio:

```text id="p4n8m2"

Aggiungi autenticazione


Planner:

3 task creati


Coder:

Modificando auth.py

```

---
