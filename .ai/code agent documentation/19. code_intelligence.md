# 1. AST Layer

## Cos'è

AST (Abstract Syntax Tree) è la rappresentazione strutturata del codice.

Esempio:

Codice:

```python
def calculate_price(value, tax):
    return value + tax
```

---

AST:

```json id="p4m8x2"
{
"type":
"Function",

"name":
"calculate_price",

"parameters":
[
"value",
"tax"
],

"return":
"expression"
}
```

---

L'agente non vede più testo.

Vede strutture.

# 4. Semantic Analysis Layer

AST dice:

"Cosa esiste"

La semantica dice:

"Cosa significa"

---

Esempio:

Due funzioni:

```python
save_user()

create_user()
```

AST:

Sono diverse.

Semantica:

Potrebbero avere responsabilità sovrapposte.

---

Analisi:

```text id="m5q8n2"

create_user

+

save_user


Possible duplication:

HIGH
```
