# Considerazione principale

La RX 6800 XT è una GPU molto valida per inferenza locale, ma la limitazione principale non è la potenza di calcolo.

È la VRAM.

16 GB permettono:

* ottimi modelli piccoli/medi;
* modelli grandi quantizzati;
* agenti multi-step.

Non permettono:

* grandi modelli 30B+ in precisione elevata;
* contesti enormi completamente caricati in GPU.

---

# Strategia corretta

Non usare un unico modello enorme.

Utilizzare:

```text id="p3v8n6"
Small Model

↓

Routing

↓

Large Model solo quando serve
```

# Principio finale

Con una RX 6800 XT il limite non è creare un AI Software Engineer.

Il limite è progettare bene il sistema.

Un singolo modello enorme non è la soluzione migliore.

Un sistema con:

```text id="v8m3q5"
modelli specializzati

+

retrieval intelligente

+

tool controllati

+

verifica automatica
```

può superare un approccio basato solo sulla dimensione del modello.
