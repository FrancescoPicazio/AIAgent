# AI Software Engineer Agent - Parte 53

# Supervisor Agent finale: orchestrazione completa, routing dinamico, gestione errori e controllo sistema multi-agente

## Obiettivo

Definire il componente centrale che coordina tutti gli agenti del sistema.

Il Supervisor Agent è il "Project Manager tecnico" dell'AI Software Engineer.

Non scrive codice direttamente.

Il suo compito è:

* capire la richiesta;
* scegliere gli agenti necessari;
* coordinare il workflow;
* controllare qualità;
* gestire errori;
* decidere quando chiedere approvazione umana.

---

# Principio fondamentale

Un sistema con singoli agenti:

```text id="m7q3x8"

Planner

Coder

Tester

Security

Research

```

non è ancora intelligente.

Serve qualcuno che sappia:

> quale agente chiamare, quando chiamarlo e perché.
