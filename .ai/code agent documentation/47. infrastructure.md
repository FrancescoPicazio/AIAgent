# AI Software Engineer Agent - Parte 47

# Local Infrastructure: Ollama, AMD RX 6800 XT, gestione modelli, quantizzazione e ottimizzazione hardware

## Obiettivo

Definire l'infrastruttura locale necessaria per eseguire l'AI Software Engineer completamente offline.

L'obiettivo è avere:

* LLM locali;
* agenti LangGraph;
* vector database locale;
* memoria persistente;
* strumenti di sviluppo;
* evaluation con LangSmith locale/self-hosted quando possibile.

---

# Principio fondamentale

Il sistema deve essere progettato considerando il limite hardware.

La GPU non determina solo "quanto è veloce".

Determina:

* quali modelli puoi usare;
* quale contesto puoi mantenere;
* quanti agenti puoi eseguire contemporaneamente.
