# AI Software Engineer Agent — ROADMAP MASTER

> Questo documento è la lista ordinata di tutti i 58 documenti di specifica del progetto.
> Ogni voce contiene: checkbox di completamento, titolo, descrizione sintetica e riferimento preciso al documento sorgente.
> L'agente in loop deve leggere ogni documento referenziato prima di implementare il task corrispondente.

---

## Come usare questa ROADMAP

1. L'agente legge questa roadmap all'avvio di ogni sessione.
2. Individua il primo task con checkbox `[ ]` non completata.
3. Apre il documento referenziato e legge **tutto il contenuto**.
4. Implementa quanto descritto nel documento.
5. Valida il risultato con i criteri di completamento indicati nel documento stesso.
6. Segna la checkbox come `[x]` e passa al task successivo.
