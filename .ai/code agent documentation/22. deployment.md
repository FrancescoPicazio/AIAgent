# AI Software Engineer Agent - Parte 22

# Deployment Agent e Infrastructure Management: ambienti, container, rilascio e monitoring

## Obiettivo

Definire il livello che permette all'agente di gestire il ciclo completo del software fino all'esecuzione.

Finora l'agente è stato in grado di:

* comprendere codice;
* progettare modifiche;
* implementare feature;
* eseguire test;
* gestire Git.

Ora deve comprendere anche dove e come il software viene eseguito.

---

# Principio fondamentale

